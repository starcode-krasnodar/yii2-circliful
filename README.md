# Circliful for Yii2
[Circliful](https://github.com/pguso/jquery-plugin-circliful) jquery-plugin integration.

Usage: 
```
<?php

use starcode\yii\circliful\Circliful;

echo Circliful::widget([
    'clientOptions'=>[
        'percent'=>75,
         'animationStep'=>5,
         ....
     ]]);
?>
```
